# Программирование корпоративных систем - Индивидульные проект "Шифровщик файлов"

---

## Суть проекта
Графическое приложение для шифрования и дешифрования файлов произвольного размера, используя реализацию алгоритма AES-256-GCM из библиотеки OpenSSL. 
Программа написана на языке C++, для создания графического интерфейса использовалась библиотека QT.
Разработка осуществлялась в IDE "Qt Creator".

---

## Использованные сторонние библиотеки
 - QT
 - OpenSSl
